# Spice-Idle

Expand your spice empire!

## Accessibility Features

Spice Idle includes the following accessibility features:

- **Screen Reader Support**: Important game events like challenge completions and new discoveries are announced to screen readers via ARIA live regions
- **Keyboard Navigation**: Full hotkey support for tab/subtab navigation and game actions (can be toggled on/off in settings)
- **Text Visibility Options**: Adjustable text visibility settings for improved readability
- **Reduce Flashing**: Option to reduce flashing animations for users sensitive to visual effects
- **ARIA Landmarks**: Proper semantic structure with tab panels, navigation, and dialog roles for assistive technology
- **Progress Indicators**: Accessible progress bars with ARIA value attributes
